# PyQuest

A responsive PHP and static quiz website containing all 73 questions from Python.pdf. The original question wording, answer areas, and answer-key values are preserved.

## XAMPP

Place this folder inside htdocs, start Apache, and open the folder through localhost. XAMPP uses index.php.

## Vercel

Import the GitHub repository into Vercel and deploy with the default settings. Vercel uses index.html and requires no build command.
