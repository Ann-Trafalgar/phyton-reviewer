# PyQuest

A responsive PHP and static quiz website containing 84 questions compiled from the supplied Python study materials. The welcome screen can launch either the complete set or only the questions added after the original 73.

## XAMPP

Place this folder inside htdocs, start Apache, and open the folder through localhost. XAMPP uses index.php.

## Vercel

Import the GitHub repository into Vercel and deploy with the default settings. Vercel uses index.html and requires no build command.
